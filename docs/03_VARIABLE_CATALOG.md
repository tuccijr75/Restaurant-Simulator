# 03 Variable Catalog

## Purpose

This file lists the major adjustable systems that Restaurant Simulator must model.

## Demand

- Customer segments.
- Arrival rates.
- Basket size.
- Order complexity.
- Patience.
- Complaint risk.
- Satisfaction.

## Channels

- Drive-thru.
- Front counter.
- Mobile pickup.
- Delivery marketplace style channel.

## POS and KDS

- Menu items.
- Modifiers.
- Combos.
- Voids.
- Refunds.
- Ticket
